# Rivet Brand Draft

Первичный знак для бренда Rivet.

Идея:
- Базовая форма собирает букву `R` из отдельных "пластин".
- Центральная стальная шляпка читается как настоящий rivet и как узел соединения.
- Диагональная ножка `R` намекает на туннель/маршрут/линк, а не просто на типографику.
- Палитра уходит в графит и медь вместо стандартного terminal-green, чтобы бренд был ближе к инструменту, а не к очередному generic devtools-логотипу.

Файлы:
- `rivet-icon.svg` — основной символ в формате app-badge.
- `rivet-logo.svg` — горизонтальный lockup с wordmark.

Палитра:
- `#0B1322` — глубокий графит.
- `#1E3148` — холодный синий сланец.
- `#FFD08A` — светлая латунь.
- `#E56541` — медный акцент.
- `#35D7CC` — холодный signal/accent для намёка на сеть.
